# Helm Charts

This folder contains the Helm charts for the Rafood API application. These charts are used to deploy the application to a Kubernetes cluster and can be used to deploy different applications to the same cluster.

## Structure

- `rafood-api/`: Helm chart for the Rafood API application.

## Notes about the usage of the Helm charts

More details about the usage of the Helm charts can be found in the [docs/local-deployment.md](../docs/local-deployment.md) file at `Deploying application with Helm` section.
