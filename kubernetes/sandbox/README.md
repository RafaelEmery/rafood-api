# Kubernetes Sandbox

This folder contains the Kubernetes sandbox for the Rafood API application. This sandbox is used to document the study notes and examples.

## Structure

- `manifests/`: Contains the raw manifests for the Rafood API application.
- `argocd/`: Contains the ArgoCD configuration for the Rafood API application.

## More details

You can find more details at directories README files at each directory:

- `manifests/`: [README.md](manifests/README.md)
- `argocd/`: [README.md](argocd/README.md)

All the notes and examples are explained and documented in the [docs/local-deployment.md](../docs/local-deployment.md) file.
