# Raw Manifests

This folder contains the raw manifests for the Rafood API application. These manifests are used to deploy the application to a Kubernetes cluster and can be used to deploy different applications to the same cluster.

## Structure

- `deployment.yml`: Simple deployment manifest for the Rafood API application.
- `service.yml`: Simple service manifest for the Rafood API application.

## Notes about the usage of the raw manifests

More details about the usage of the raw manifests can be found in the [docs/local-deployment.md](../docs/local-deployment.md) file at `Basic service and deployment configuration example` section.
